# Project Rules

## Scripts
Anytime you create a script, also create `input.json` and `output.json` alongside it. These files should be updated every time the script runs to give an overview of what it just did:
- `input.json` — the parameters/config the script ran with (date range, channels, reps, etc.)
- `output.json` — a summary of what happened (leads found per channel, per rep, any errors, timestamp)
